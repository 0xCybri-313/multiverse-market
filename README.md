# Multiverse Market 🛒

A modern eCommerce app built with React, Vite, and Tailwind CSS.

## 🔗 Links

- Live Preview: https://github.com/0xCybri-313/multiverse-market.git
- Clone Repo: https://github.com/0xCybri-313/multiverse-market.git

## ⚙️ Tech

React + Vite • Tailwind CSS • Context API • Custom Hooks  
React Router (params) • Clerk Auth • API integration

## ✨ Features

- Product listing from API
- Dynamic product pages
- Global state management
- Authentication
- Responsive UI

## 🚀 Run Locally

```bash
git clone https://github.com/your-username/multiverse-market.git
cd multiverse-market
npm install
npm run dev
```
